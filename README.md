# Overview

The **Embedded Systems Learning Lab** serves as the foundation of my embedded systems engineering journey.

This repository contains a growing collection of projects developed while learning the fundamentals of embedded systems, robotics, wireless communication, sensor interfacing, automation, and IoT.

Rather than focusing on a single application, this repository documents my progression from introductory hardware experiments to increasingly sophisticated embedded systems integrating multiple sensors, communication protocols, and control techniques.

Many of the concepts explored here later evolved into larger engineering projects featured throughout my GitHub portfolio, including healthcare embedded systems, FPGA development, digital control systems, and research projects.
