# 🔬 Embedded Systems Learning Lab

> A curated collection of embedded systems projects documenting my learning journey in Arduino, ESP8266/ESP32, sensors, actuators, IoT, and hardware interfacing.

---

# 📖 Overview

The **Embedded Systems Learning Lab** serves as the foundation of my embedded systems journey.

This repository contains a growing collection of projects developed while learning the fundamentals of embedded systems engineering, including microcontroller programming, sensor interfacing, wireless communication, automation, and IoT.

Rather than focusing on a single application, this repository documents the progression from beginner experiments to increasingly complex embedded system designs.

Many of the concepts explored here later evolved into larger engineering projects featured in my portfolio.

---

# 🎯 Objectives

- Learn microcontroller programming
- Understand digital and analog interfacing
- Explore sensor integration
- Build IoT applications
- Develop hardware debugging skills
- Practice embedded C/C++
- Build confidence with real hardware
